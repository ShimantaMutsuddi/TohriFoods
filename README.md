## 🛠️ Installation
1️⃣ **Clone the repository**
```sh
git clone https://github.com/ShimantaMutsuddi/TohriFoods.git
cd tohri_foods
```
2️⃣ **Install dependencies**
```sh
flutter pub get
```
3️⃣ **Run the app**
```sh
flutter run
```
